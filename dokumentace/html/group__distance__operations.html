Vypočítá vzdálenost mezi dvěma clustery pomocí vzdáleností jednotlivých objektů v obou clusterech pomocí eukleidovské vdálenosti(trojúhelník)

Parameters
c1první cluster, který obsahuje objekty ze kterých se počítá vzdálenost s objekty obsaženými v druhém clusteru. Porovnává se každý objekt s každým a nejmenší hodnota je výsledná hodnota
c2druhý cluster, který obsahuje objekty ze kterých se počítá vzdálenost s objekty obsaženými v prvním clusteru. Porovnává se každý objekt s každým a nejmenší hodnota je výsledná hodnota
Precondition
c1 nesmí být nulový pointer, jinak se program ukončí
velikost c1 musí být větší než 0, jinak se program ukončí
c2 nesmí být nulový pointer, jinak se program ukončí
velikost c2 musí být větší než 0, jinak se program ukončí
Postcondition
změří se nejmenší vdálenost, která výjde z počítaní vzdáleností jednotlivých objektů v clusterech c1 a c2... Porovnává se každý s každým
Returns
vrátí se vzdálenost dvou zadaných clusterů v datovém typu "float"
