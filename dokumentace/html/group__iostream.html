int load_clusters ( char *  filename,
struct cluster_t **  arr 
)

Funkce načítá a inicializuje objekty a přidává je do clusterů čtením ze vstupního souboru. Pro každý objekt se vytvoří cluster, do kterého se objekt přidá. Clustery se poté vloží do pole clusterů

Parameters
filenameJedná se o název souboru, ze kterého se budou načítat data. První řádek má podobu "count=N", kde N je číslo vyjadřující počet řádků, se kterýma se má pracovat. Další řádky obsahují data o objektech ve formátu ID X Y
arrJedná se o pole clusterů, do kterého se budou ukládat jednotlivé clustery
Precondition
Soubor filename by měl jít otevřít, jinak program vyhodí chybu a vypne se
Pole clusterů arr by mělo být definované už před začátkem funkce
Postcondition
Na konci funkce bude pole clusterů obsahovat všechny vytvořené clustery, které budou obsahovat všechny načtené objekty
Načte se jen takový počet řádků, jaká je hodnota čísla na prvním řádku. Každý cluster bude obsahovat jeden objekt
Returns
Fuknce vrátí hodnotu vyjadřující počet načtených objektů v datovém typu "int"
